================================================================================
ЧТО НУЖНО СДЕЛАТЬ В LM STUDIO ДЛЯ РАБОТЫ API
================================================================================

ПРОБЛЕМА: API возвращает 502, хотя модель работает в Chat.

РЕШЕНИЕ:

1. Откройте LM Studio
2. Нажмите Settings (Настройки) -> Local Server (Локальный сервер)

3. ПРОВЕРЬТЕ:
   - Кнопка "Start Server" должна быть нажата
   - Должно быть написано "Server running"
   - Порт должен быть указан (обычно 1234)

4. ОТКЛЮЧИТЕ:
   - "Load model on demand" (Загрузка модели по требованию)
   - "Automatic unloading of unused models" (Автоматическая разгрузка)

5. ПЕРЕЗАПУСТИТЕ:
   - Нажмите "Stop Server"
   - Подождите 5 секунд
   - Нажмите "Start Server"
   - Дождитесь статуса "READY" для модели

6. ПРОВЕРЬТЕ:
   - В разделе Local Server модель должна быть "READY"
   - В Chat модель должна отвечать (это уже работает)

7. ПОСЛЕ НАСТРОЙКИ:
   - Запустите: python FINAL_CONNECTION_TEST.py
   - Или запустите: python test_agent_direct.py

================================================================================
ВАЖНО: Chat и API - это разные вещи!
Модель может работать в Chat, но не быть доступна через API,
если Local Server не настроен правильно.
