📋 ЧТО НУЖНО СДЕЛАТЬ:

1. УСТАНОВИТЕ LM STUDIO (если еще не установлен)
   → Скачайте: https://lmstudio.ai
   → Установите и запустите

2. СКАЧАЙТЕ МОДЕЛЬ
   → Откройте LM Studio
   → Перейдите в раздел "Search"
   → Найдите "deepseek-coder" или "CodeLlama"
   → Нажмите "Download"

3. ЗАГРУЗИТЕ МОДЕЛЬ
   → Перейдите в раздел "Chat"
   → Нажмите "Select a model to load"
   → Выберите скачанную модель
   → Нажмите "Load Model"

4. ВКЛЮЧИТЕ LOCAL SERVER
   → Перейдите в "Settings"
   → Найдите "Local Server"
   → Включите переключатель
   → Нажмите "Start Server"
   → Должно появиться "Server running"

5. В GUI ИНТЕРФЕЙСЕ:
   → Выберите провайдер "lmstudio" в боковой панели
   → Нажмите "Обновить список моделей"
   → Выберите загруженную модель
   → Готово! Можете использовать!
