ПРОБЛЕМА: Все запросы возвращают 502 Bad Gateway

ПРИЧИНА: Настройка "Загрузка модели по требованию" включена

РЕШЕНИЕ:

1. Откройте LM Studio
2. Settings → Local Server
3. Найдите "Загрузка модели по требованию"
4. ОТКЛЮЧИТЕ этот переключатель
5. Найдите "Автоматическая разгрузка неиспользуемых моделей"
6. ОТКЛЮЧИТЕ этот переключатель
7. Перезапустите Local Server (Stop → Start)
8. Дождитесь загрузки модели (30-60 секунд)
9. Попробуйте снова

АЛЬТЕРНАТИВА:

Если модель работает в Chat, но не через API:
1. Откройте Chat в LM Studio
2. Отправьте любое сообщение модели
3. Дождитесь ответа
4. Это загрузит модель в память
5. После этого API должен работать
